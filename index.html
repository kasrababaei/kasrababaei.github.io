Visual Catalog App

VC App previews products of Habibi Interior Store. Users can navigate through the various categories of products and save their favourites or request quotation.
Some of the features and technologies used in this project:

  • 90% of the UI is made programmatically.
  • The data is fetched from the server in JSON format and parsed using JSDONDecoder.
  • By using CoreData, all the details of items and the user selects as favourite are saved.
  • Images are cached on the device to avoid repeatitive downloading.
My role: sole iOS developer

Public Reporting System

PRS allows users to send reports under various categories to the closest authority (based on the location of the repot) in Malaysia. Authorities will assign a contractor to the reported issue (with real time tracking).
Some of the features and technologies used in this project:

  • The UI is made 100% programmatically.
  • Secured connection is implemented using OAuth.
  • The data is fetched in JSON format using URLSession and parsed by JSONDecoder.
  • Apple Push Notification and Firebase.
  • MapKit.
  • Google APIs are used for place predictions while searching locations and fetching coordinates.
My role: sole iOS developer

  • AppStore
  • Contractor

ICON V3

ICON V3 allows users to control products and appliances in their homes using iPhones or iPads. I was the sole front-end developer of this project.
Some of the features and technologies used in this project:

  • Secured connection is established using OAuth.
  • The data is fetched in JSON format using URLSession.
  • The UI is made programmatically and also using Storyboard.
  • Apple Push Notification and Firebase.
  • MobileVLCKit library for showing live footage from CCTV cameras.
